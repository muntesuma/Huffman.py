Это моё 1ое использование GitHub в лабораторной работе
Для выполнения лабораторной работы мне нужно разбить её на несколько подзадач:
1.1) main.py открывает (на чтение в текстовом режиме) входные данные - файл code.txt (с текстом на русском или английском языке)
1.2) main.py создает строку str_code и копирует туда всё содержимое code.txt
2.1) main.py подсчитывает частоту появления каждого символа в str_code
2.2) main.py создет очередь с символами с наименьшей частотой
3.1) main.py создает бинарное дерево Хаффмана используя очередь
3.2) main.py создавет таблицу Хаффмана используя дерево
4.1) main.py использует таблицу Хаффмана для кодирования code.txt
4.2) main.py записывает закодированное сообщение в файле code_haffmaned.txt
5.1) main.py создает строку str_code_haffmaned и копирует туда всё содержимое code_haffmaned.txt
5.2) main.py использует дерево Хаффмана для декодирования str_code_haffmaned
5.3) main.py записывает декодированное сообщение в файле code_dehaffmaned.txt
